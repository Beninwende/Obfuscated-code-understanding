**Exercise Title: Product Computation with Limits**

**Problem Statement:**

Write a program that takes a list of integers and computes the product of all the integers in the list. If the product exceeds a specified limit, the program should return a distinct value indicating that the limit has been exceeded.

**Requirements:**

1. Implement a function that receives two inputs:
   - A list of integers.
   - An integer representing the upper limit for the product.
   
2. Compute the product of all integers in the input list.

3. If the product of the integers exceeds the given upper limit, return a special indicator value (e.g., -1).

4. If any integer in the list is zero, the product should immediately be calculated as zero regardless of the other values.

5. The function should handle edge cases like an empty list or lists with a single element appropriately.

6. Ensure the solution is efficient enough to handle large lists of integers.

Focus on developing a program that effectively handles multiplication and understands conditions to limit arithmetic overflow within the specified constraints.
