Exercise Title: Character Case Checker Problem Statement: You are given a single English letter, which can either be uppercase (A-Z) or lowercase (a-z). Your task is to determine the case of the letter and print a corresponding result: If the letter is uppercase, print the letter 'A'. If the letter is lowercase, print the letter 'a'. Requirements: Write a program that reads a single letter as input. The program should check whether the letter is uppercase or lowercase. Based on the check, the program should output 'A' if the letter is uppercase, or 'a' if it is lowercase. Constraints: The input will always be a single English letter (either uppercase or lowercase).
