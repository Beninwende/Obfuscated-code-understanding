Title: Counting Prime Numbers Problem Statement: Write a Java program that counts how many prime numbers are present in a given set of integers. The program should first take an integer input representing the total number of integers to be checked. Then, it should read each integer, determine if it is a prime number, and keep a count of all prime numbers found. Finally, the program should output the total count of prime numbers. Requirements: Implement a method isPrime that takes an integer as input and returns true if the number is prime, otherwise false. Implement a method run that handles user input, uses the isPrime method to check each number, and counts the prime numbers. Use a loop to process each input number and a conditional statement to update the count of prime numbers. Output the total count of prime numbers at the end. Ensure the program is efficient and handles edge cases (e.g., numbers less than 2). This exercise focuses on fundamental programming concepts such as methods, loops, conditionals, and basic input/output operations.
