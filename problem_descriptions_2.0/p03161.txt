Exercise: Minimum Jump Cost Problem Statement: You are given a sequence of stones, each with a height. A frog starts on the first stone and needs to jump to the last stone by making jumps between nearby stones. The frog can jump from stone i to any stone j such that i < j ≤ i + K, where K is a maximum jump distance. Each jump from stone i to stone j incurs a cost equal to the absolute difference in their heights, i.e., |h[i] - h[j]|, where h[i] is the height of stone i. Your task is to determine the minimum possible total cost for the frog to reach the last stone. Requirements: Input: The first line contains two integers: N (the number of stones) and K (the maximum jump distance). The second line contains N integers representing the heights of the stones: h[1], h[2], ..., h[N]. Output: Print the minimum total cost the frog needs to incur to reach the last stone. Goal: Find an efficient solution that calculates the minimum cost for the frog to jump from the first stone to the last stone, using dynamic programming.
