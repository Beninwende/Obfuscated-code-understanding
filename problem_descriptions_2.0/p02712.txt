**Exercise Title: Sum of Non-Divisibles**

**Problem Statement**

The task is to compute the sum of all integers in a given sequence from 1 to N that are not divisible by certain specified divisors. Specifically, if an integer in this sequence is divisible by any of the specified divisors, it should be excluded from the sum calculation. Your program should efficiently handle and compute this sum for large values of N.

**Requirements**

1. Read an integer N from standard input, where 1 ≤ N ≤ 10^6. This integer represents the upper bound of the sequence starting from 1.
  
2. Given two divisors, D1 and D2 (e.g., 3 and 5), investigate each integer i in the sequence from 1 to N:
   - If i is divisible by D1 or D2, skip it.
   - If i is not divisible by either D1 or D2, include it in the sum.
   
3. Calculate the sum of all integers that meet the above condition (not divisible by either D1 or D2).

4. Print the resulting sum.

This exercise helps practice loops, conditional statements, and handling large sums efficiently to avoid overflow. Ensure that the solution scales well with the size of N.
