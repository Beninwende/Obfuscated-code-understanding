**Exercise Title: Find the Minimum Steps in a Number Grid**

**Problem Statement:**

You are navigating an infinite grid of numbers. Each cell in the grid contains a value determined by the product of its row number and column number. Specifically, the cell at position (i, j) contains the value i * j. Begin your journey at the top-left corner of the grid, which is (1, 1), and your goal is to reach a grid cell containing a specified target number.

To successfully reach your target, you can move to either the next row or the next column from your current position. Your task is to determine the minimum number of steps needed to travel to any cell that contains the target number. Each move, either downwards or rightwards, counts as one step.

**Requirements:**

1. Implement a function that, given an integer **N**, represents the target number on the grid.
2. The function should calculate and return the minimum number of moves required to reach any cell containing **N** from the starting position (1, 1).
3. Your solution must efficiently handle very large values of **N**, up to **10^12**. 

Considerations:
- You begin at (1, 1) which corresponds to the value 1. 
- You can only move to the next row (i+1, j) or next column (i, j+1) in a single move.
- Find the cell with the value **N** that can be reached using the fewest moves.
