**Exercise Title: Combine Characters from Two Strings**

**Problem Statement:**

Write a program that takes two strings of equal length and combines them into a new string. The new string should be generated by sequentially taking one character from the first string and then one character from the second string, continuing this pattern until all characters from both strings are used.

**Requirements:**

1. Define a function that accepts three parameters:
   - An integer `N`, representing the length of the strings.
   - A string `A` composed of lowercase English letters of length `N`.
   - A string `B` composed of lowercase English letters of length `N`.

2. The function should return a new string created by:
   - Taking the first character of `A`, then the first character of `B`.
   - This pattern should continue until the last character of both strings is included in the new string.

3. Ensure that the inputs are valid according to the following constraints:
   - The integer `N` should be at least 1 and at most 100.
   - The lengths of both strings `A` and `B` should be exactly `N`.

4. The function should concatenate characters efficiently and output the final combined string. Avoid using any built-in functions that directly solve the problem for educational purposes. 

Focus on the concept of iteration and string manipulation to achieve the desired result. Ensure your function is robust and tested against edge cases such as strings with repeated characters and lengths at the limit of the constraints.
