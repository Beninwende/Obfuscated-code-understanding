Exercise Title: Count Consecutive 'R' Characters Problem Statement You are given a string of length 3, where each character is either 'S' or 'R'. Your task is to find the maximum number of consecutive 'R' characters in the string. Requirements Write a function that takes a string and returns the maximum number of consecutive 'R' characters in that string. The string will have exactly 3 characters, and each character will be either 'S' or 'R'.
