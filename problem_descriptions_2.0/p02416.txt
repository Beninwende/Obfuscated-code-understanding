Exercise: Sum of Digits Problem Statement: Write a program that reads multiple lines of input, where each line contains an integer. For each integer, your program should calculate and print the sum of its digits. The input ends when a line with the number 0 is encountered. Do not process or output anything for this terminal symbol. Requirements: Your program should read and process multiple integers, one per line. For each integer, calculate the sum of its digits. Stop reading further inputs when the number 0 is encountered. The number of digits in any input integer does not exceed 1000. Functions to Implement: A function to read an integer from input. A function to compute the sum of the digits of an integer. A function to print the result for each integer. Ensure that the program terminates when 0 is encountered. Focus on using basic control structures like loops, conditionals, and data manipulation techniques such as converting the integer to a character array for easier access to individual digits.
