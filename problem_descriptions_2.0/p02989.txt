### Exercise Title: Equal Split of Sorted Numbers

#### Problem Statement:

You are given an array of integers that represent a series of numbers. Your task is to find a way to split this array into two equal parts by sorting the array and selecting an appropriate splitter value. The splitter value should be chosen such that the number of elements strictly less than the splitter is equal to the number of elements greater than or equal to the splitter.

Your goal is to determine how many unique splitter values exist that will allow the array to be divided into such equal parts.

#### Requirements:

1. Write a function that takes an array of integers as input.
2. Sort the given array in non-decreasing order.
3. Calculate the number of unique values that can be used as the splitter value to evenly divide the array.
4. Return the count of such unique splitter values.

Note:
- The input array will have an even number of elements.
- Consider only the values present in the array as potential splitters.
- Implement the provided function to achieve this goal efficiently, keeping in mind the complexity constraints.

This exercise focuses on understanding basic sorting, array manipulation, and logical partitioning of data.
