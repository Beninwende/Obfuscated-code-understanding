Exercise Title: Common Divisors Problem Statement: You are given two positive integers, X and Y. Your task is to find the N-th largest positive integer that divides both X and Y. You need to: Identify all the common divisors of X and Y. Sort these divisors in descending order. Return the N-th largest divisor. Requirements: Implement a function to compute the common divisors of two given integers. The function should return the N-th largest divisor of both integers. Ensure the divisors are sorted in descending order. Functions: Input: Two integers X and Y (1 ≤ X, Y ≤ 100). An integer N indicating which largest divisor to find (1 ≤ N ≤ number of common divisors). Output: The N-th largest common divisor of X and Y.
