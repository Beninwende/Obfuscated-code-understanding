Title: Compute GCD and LCM of Two Numbers Problem Statement: Create a program that calculates the greatest common divisor (GCD) and the least common multiple (LCM) of two integers. The program should process multiple pairs of integers, each pair provided on a new line. It should compute and display the GCD and LCM for each pair. Requirements: Implement a function to calculate the GCD of two integers using an efficient algorithm, such as the Euclidean algorithm. Implement a function to calculate the LCM of two integers using the relationship between GCD and LCM. Read multiple pairs of integers from the input until the input ends (EOF). Each pair will be separated by a single space. For each pair, compute the GCD and LCM, ensuring the results adhere to the constraints. Print the GCD and LCM for each pair on a single line, separated by a space. Constraints: Both integers will be positive. The range of each integer is sufficiently large to test your program's efficiency and correctness. Ensure your program handles up to a defined maximum number of pairs correctly and efficiently.
