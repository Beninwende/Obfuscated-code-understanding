Exercise: Insertion Sort with Intermediate Trace Problem Statement: Write a program that implements the Insertion Sort algorithm to sort a sequence of integers in ascending order. During the sorting process, the program should trace and print the array after each step, illustrating how the elements are rearranged. The Insertion Sort algorithm works by iterating through the array and inserting each element into its correct position within the sorted portion of the array. Requirements: The program must take an integer 𝑁 N as input, representing the number of elements in the sequence. It should then take 𝑁 N space-separated integers as input, which represent the unsorted sequence. The program should sort the sequence using the Insertion Sort algorithm. After each step of the insertion process, the program must output the current state of the array, showing the intermediate results. The sorted array must be printed at the end of the sorting process. You should implement the following functions: sort(int[] arr): This function should perform the Insertion Sort and print the array after each insertion step. printArray(int[] arr): This helper function will output the array with space-separated values, displaying the current sequence after each step.
