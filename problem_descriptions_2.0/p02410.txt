Exercise Title: Matrix-Vector Multiplication Problem Statement: Write a program that performs matrix-vector multiplication. You are given a matrix A of size n x m and a column vector b of size m x 1. The goal is to compute the product of matrix A and vector b, which will result in a new column vector c of size n x 1. Each element of c is computed as the dot product of the corresponding row of A and the vector b. Requirements: The program must first read the dimensions of the matrix (n rows and m columns) and the matrix itself. Then, it must read the vector b of size m x 1. Key Concepts: Matrix representation and multiplication. Iteration through matrix rows and vector elements. Understanding dot products in linear algebra.
