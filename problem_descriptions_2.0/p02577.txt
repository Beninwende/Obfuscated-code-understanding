Exercise Title: Check If a Number is a Multiple of 9 Problem Statement: You are given a very large integer 𝑁 N. An integer 𝑁 N is a multiple of 9 if and only if the sum of its digits is divisible by 9. Your task is to determine whether the integer 𝑁 N is a multiple of 9. Requirements: You need to process the digits of the number 𝑁 N and compute the sum of the digits. Check if the sum of the digits is divisible by 9. If the sum is divisible by 9, print 'Yes'. Otherwise, print 'No'. Constraints: 0 ≤ 𝑁 < 1 0 200000 0≤N<10 200000 𝑁 N is an integer.
