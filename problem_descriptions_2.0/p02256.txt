Title: Greatest Common Divisor (GCD) Calculation Problem Statement: Write a Java program that calculates the greatest common divisor (GCD) of two integers using the Euclidean algorithm. The program should take two integers as input and output their GCD. Requirement: Implement a method named gcd that takes two integer parameters and returns their GCD using recursion. The gcd method should follow the Euclidean algorithm: If the second number is 0, return the first number as the GCD. Otherwise, recursively call the gcd method with the second number and the remainder of the division of the first number by the second number. In the main method, read two integers from the user, call the gcd method with the larger number as the first argument and the smaller number as the second argument, and print the result. Ensure the program is structured clearly and handles basic input/output operations. This exercise focuses on practicing recursion, method implementation, and basic input/output in Java.
