### Exercise Title: Sort and Identify Top Three Numbers

#### Problem Statement:
You are given a list of ten integers. Your task is to write a program that identifies and displays the three largest numbers from this list in descending order. These three numbers represent the highest values among the given inputs.

#### Requirements:
1. Write a function that takes a list of ten integers as input. Each integer will be in the range of 0 to 10,000 (inclusive).
2. Sort the integers in descending order based on their value.
3. Extract the top three integers from the sorted list.
4. Print these three integers, each on a new line, starting with the largest and ending with the third largest.
5. Ensure the program handles ten input integers correctly, adhering to the constraints specified.

This exercise practices fundamental programming concepts such as array manipulation, sorting algorithms, and basic input/output operations.
