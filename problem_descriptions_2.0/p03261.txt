Title: Count Valid Sequences Problem Statement: Write a program that validates a sequence of words based on two rules: Each word must start with the last character of the previous word. No word should be repeated in the sequence. The program should read the number of words and the sequence of words, then determine if the sequence satisfies the rules. If it does, print 'Yes'; otherwise, print 'No'. Requirements: Use a HashMap to track used words and ensure no duplicates. Implement helper methods to read and parse input efficiently. Ensure the program handles input dynamically and checks the sequence iteratively. Focus on fundamental programming concepts such as loops, conditionals, and data structures. This exercise practices input handling, string manipulation, and the use of data structures for validation.
